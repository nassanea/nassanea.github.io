解題思路

使用 ArrayList 當成「新增」、「移除」的資料結構會使這一題的時間複雜度變成 $O(N^2)$，把資料結構調整成新增和刪除操作都很快速的 Set 後，可以改善時間問題，程式碼如下：

特性 舊版本 (ArrayList) 新版本 (HashSet)
查詢 (contains) $O(N)$ (要從頭掃描) $O(1)$ (直接定位)
刪除 (remove) $O(N)$ (要掃描 + 移動元素) $O(1)$ (直接拿走)
整體時間複雜度 $O(N^2)$ (很慢) $O(N)$ (前面迴圈) + $O(N \log N)$ (最後排序)
空間複雜度 $O(N)$ $O(N)$
適用場景 資料量極小 (N < 5000) 通用解法，可處理大數據
