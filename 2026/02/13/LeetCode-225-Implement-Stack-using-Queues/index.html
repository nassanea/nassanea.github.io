題目：225. Implement Stack using Queues

問題描述

用最多兩個 Queue 去實作出一個 stack

解題思路

本題有兩種解法，做過 詳見 232. Implement Queue using Stacks 的話首先會想到用兩個 queue 模擬出 stack 的概念，這也是可行的，但其實這題只需要一個 queue 就可以模擬出 stack 了

因為 push 已經負責維護好 LIFO 的原則，而 pop 就只需要簡單的從 queue 的 front 取出資料即可的到 MyStack 最頂端的資料了

其實也有使用兩個佇列來模擬 stack 的方法
這就涉及記憶體位址的運用了

push 只需要普通的 push 到 queue 的 back 即可，不用特別處理
