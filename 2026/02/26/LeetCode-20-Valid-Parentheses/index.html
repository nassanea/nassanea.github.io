LeetCode-20. Valid Parentheses

題目：20. Valid Parentheses

參考來源：20. 有效的括号

問題描述

要判斷字串 s 是否為有效的括號，有效的括號必須要有配對且括號類型需一致

解題思路

本題需要用到 stack 的特型來解題，概念是用回圈遍歷字串一一檢查每個括號：

  • 當看到左括號 (, [, { 時，就把相對應的右括號 push 進入堆疊
  • 當看到右括號時，檢查堆疊的 top 的括號，如果跟當前遍歷到的括號相同 c[i]==stack.top()，代表成功配對，就可以把堆疊上層的該括號取出（pop）
    • 如果看到右括號時卻發現堆疊沒有元素（也就是無法得出 top），代表這個右括號前面根本不存在可與其配對的左括號，直接回傳 false
  • 接著就繼續檢查下一個括號直到遍歷完整個字串 s
    當回圈結束時檢查堆疊是否為空：
  • 空：代表每個括號都有成功配對到，回傳 true
  • 不為空：有非法括號，回傳 false

其他：

  • 以及當字串長度不為偶數時，可以直接回傳 false，因為奇數無法配對，必定不是有效括號。
  • 看到左括號時卻 push 右括號進入 stack，是因為這樣比較好檢查右括號的部分是否有符合
