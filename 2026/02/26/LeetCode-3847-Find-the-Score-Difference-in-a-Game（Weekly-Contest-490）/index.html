LeetCode-3847. Find the Score Difference in a Game（Weekly Contest 490）

題目：3847. Find the Score Difference in a Game

問題描述

給予一個分數陣列 nums[]，nums[i] 代表第 i 輪比賽的比賽分數，要根據以下規則把該分數給予兩位選手的其中一位，最後回傳的 first - second 選手的分數差
遊戲把兩位選手分成主動方和被動方，初始狀態會設定第一位選手為主動方，因此我們做以下設定

  • 預設的第一位選手（主動方）設為 first
  • 另一位是 second

規則是

  • nums[i] 為 odd，那攻守交換
  • nums[i] 為 even，攻守不換
  • i 為每週期第六場比賽時，會再進行一次攻守交換，所以 i 是 5, 11, 17,…
  • 第 i 場比賽的分數 nums[i] 會由主動方收下
  • 要回傳的分數差是 first - second

解題思路

解法是用一個 boolean 值去判斷當前主動方（active）是哪個人：

  • true 代表 active 是 first
  • false 代表 active 是 second

會有以下四種情況，只有其中的兩種最終需要換手

odd 6th swap?
T F T
F T T
T T F
F F F

所以程式邏輯的方法為

  1. 設置布林值判斷當前 active
  2. 遍歷 nums[] 陣列
  3. 判斷 nums[i]%2!=0 時為 odd 且 (i+1)%6!=0 不為 6th 比賽，會換手一次，或是 nums[i]%2!==0 時為 even 且 (i+1)%6==0 為 6th 比賽，也會換手一次
  4. 若非上述情況，則不換手
  5. 換手邏輯處理完，即可 assign 分數給 first / second 玩家
  6. 回傳分數差

這樣解法的時間複雜度是 $O(n)$；空間複雜度是 $O(1)$
